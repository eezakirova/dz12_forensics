## Описание задания

Загрузить дамп памяти или дамп сети.  
Можно использовать предоставленный дамп сети или найти тестовые дампы сети / памяти из открытых учебных источников.

Необходимо найти ключевые артефакты:
- процессы,
- соединения,
- DNS-запросы.

---

## Этап 1. Загрузка данных

Для выполнения работы использован сетевой дамп (PCAP-файл).

Дамп загружен с помощью библиотеки `pyshark` в среде Python.

---

## Этап 2. Извлечение ключевых артефактов

В рамках анализа сетевого дампа выполнено:

- Выделение DNS-запросов  
- Определение доменных имён  
- Подсчёт количества обращений к каждому домену  

Извлечённые данные сохранены в файл:

- `top_10_domains.csv` — топ-10 наиболее часто запрашиваемых доменов

---

## Этап 3. Визуализация результатов

Построена минимальная визуализация:

- График количества DNS-запросов по времени  
- Сохранён файл `dns_requests_over_time.png`

Для визуализации использована библиотека `matplotlib`.

---

## Используемые инструменты

- Python 3  
- pyshark  
- pandas  
- matplotlib
